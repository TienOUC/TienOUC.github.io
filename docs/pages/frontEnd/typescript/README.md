## 1. 简介

关于 `TypeScript` 的一些基础概念。

- 微软开发的开源编程语言，是 `JavaScript` 的超集
- 能够在编写阶段检查错误（静态类型检查）
- 语法提示更丰富
- 适合大型开发

**增加的功能:**
:::tip

1. 类型批注和编译时类型检查
2. 类型推断
3. 类型擦除
4. 接口
5. 枚举
6. Mixin
7. 泛型编程
8. 名字空间
9. 元组
10. Await

:::

## 3. 安装使用

```shell
1. npm安装
npm install -g typescript

2. 编写.ts 文件
app.ts

3. 执行tsc命令转换
tsc app.ts

4. 执行转换后的.js文件
node app.js
```
